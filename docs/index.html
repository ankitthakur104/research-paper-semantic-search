Retrieval Pipeline

SPECTER2 domain-specific embeddings (trained on citation graphs) outperform generic models for scientific text. Pinecone stores 50K paper embeddings with metadata filters (year, venue, citations).
AI Summaries

GPT-4o generates a 2-3 sentence research synthesis for each retrieved paper, highlighting relevance to the query. Includes method, dataset, and key result extraction.
