Search Pipeline

1. Encode query — Sentence-BERT (all-MiniLM-L6)
2. Pinecone search — top-5 cosine ANN
3. Re-rank — cross-encoder reranking
4. Summarize — GPT-4o per paper
Ingestion Pipeline (n8n)

Runs nightly at 2AM IST. Fetches new arXiv CS/AI papers, chunks abstracts into 256-token segments, embeds and upserts to Pinecone. ~300-500 new papers/day ingested automatically.
